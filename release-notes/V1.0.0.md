# Failure Simulator v1.0.0 - Release Notes

The initial stable release of the modernized Failure Simulator for GeoFS v3.9.

## 🚀 Key Features
- **Strict Legacy Logic**: Fully migrated the mechanical rules from the original GeoFS Failures (GGamerGGuy) script.
- **Engine Smoke FX**: Realistic black smoke particles for engine failures using custom emitters.
- **Timed Fuel Leaks**: Implemented the 2-minute "limp" period followed by total engine shutdown.
- **Multi-Bogie Gear Support**: Advanced detection for complex aircraft (A380, 747) ensuring all wheels collapse correctly.
- **Integrated FIX System**: Granular recovery buttons for every individual system.

## 🎨 UI & UX
- **Glassmorphic Design**: Sleek, transparent UI that matches the GeoFS v3.9 Addon-Pack aesthetic.
- **Global Controls**: "FAIL ALL" and "FIX ALL" buttons for rapid scenario setup.
- **Category Toggles**: Micro-controls `[X]` and `[✓]` for entire systems (Engines, Hydraulics, etc.).
- **Probability Simulator**: Background failure loop with automatic input locking for safety.

## 🛠️ Reliability Improvements
- **100% Notification Coverage**: Every failure and recovery triggers a professional, punctuated alert.
- **Robust Fallbacks**: Added spatial coordinate detection to support aircraft with non-standard suspension names.
- **Undefined Safety**: Included checks to prevent script crashes on gliders or non-engine aircraft.
